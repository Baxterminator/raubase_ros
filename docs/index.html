    Welcome to the Raubase-ROS package documentation

    This website hosts the documentation about the package raubase_ros, a translation from the DTU Raubase software.

    The original author of the code is Jens Christan Andersen, and the ROS2 conversion was made by Geoffrey Côte (aka Meltwin online).

    This project is distributed under the MIT license. Copyright © 2017 -2024 by DTU

    Maintainer

    This package is maintained by:

    • Geoffrey Côte ECN (General Engineering - Robotics) & DTU (MSc Autonomous Sytem) student
    • Jens Christan Andersen Associate professor in the Department of Electrical and Photonics Engineering (Automation and control)

    Dependencies

    This projects runs on:

    • ROS: ROS2 Humble,
    • Python: 3.10
    • OS: Ubuntu 22.04,
    • Arch: Tested Raspberry Pi 4 and x86 computer,
